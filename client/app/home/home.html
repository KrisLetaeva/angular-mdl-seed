Sass + SMACSS

Sass is the most mature, stable, and powerful professional grade CSS extension language. The project is structured following the SMACSS architecture. Write your CSS in a modular and scalable way, the build system will compile your .scss files into a single css files. It should be easy to integrate less, stylus or any other preprocessor if you prefer. Pure is the default CSS framework, by you can easily plug your own.

Keep Your Code Reusable

Every general purpose directive, service or filter, should be placed into the common directory, in this way you can copy and paste the directory into another project, require the module you need, and you are ready to go with your new project.
